### Custom Mapping
- < C-\ >: toggling nerdtree
- < C-up >: split tab
- < C-down >: close current tab
- < C-left >: previous tab
- < C-right >: next tab

### Installation
1. Clone repo into ~/
2. Delete all directories in .vim/bundle/
3. Clone everything from plugins and files support into .vim/bundle
